## Short Description (About)

## Extended Description (for README or repo intro)
CabLog is an open-source operations tool that converts handwritten cab trip sheets and receipts into structured, reviewable records.

It combines AI-based extraction, human review, status tracking, and monthly export workflows so transport and admin teams can reduce manual entry errors and improve processing speed.

The project supports both cloud AI (Gemini) and local AI (LM Studio via OpenAI-compatible server), allowing teams to choose between maximum convenience or privacy-first local inference.

## Professional Highlights (Resume/Recruiter Friendly)
- Designed a provider-agnostic AI extraction pipeline (cloud + local model support)
- Implemented queue-based asynchronous processing with progress tracking and retry-safe flows
- Built schema-first OCR normalization for reliable downstream exports
- Delivered human-in-the-loop review UX for quality control before finalization
- Enabled multi-format business exports (CSV/Excel) for accounting and reporting
